<h2>Challenge</h2>

  <p>
   O ForumHub é uma API REST desenvolvida com Java e Spring Boot, que simula o funcionamento de um fórum de discussões.
   A aplicação permite o cadastro de usuários, autenticação via JWT e gerenciamento de tópicos relacionados a cursos.
   O projeto foi construído seguindo boas práticas de arquitetura em camadas, separando responsabilidades entre controllers, services, repositories e DTOs.
   Esse projeto faz parte da formação em java do programa ONE com a Alura + Oracle.
  </p>

<h2>Objetivos do projeto</h2>
<ul>
<li>Desenvolvimento de APIs REST</li>
<li>Autenticação e autorização com JWT</li>
<li>Estruturação de aplicações backend profissionais</li>
<li>Boas práticas com Spring Boot</li>
</ul>

<h2>Tecnologias utilizadas</h2>

  <li>Spring Data JPA/Hibernate</li>

<h2>Funcionalidades</h2>

  <li>Cadastro de usuários</li>
  <li>Cadastro de Tópicos</li>
  <li>Autenticação com geração de Token JWT</li>
  <li>Deletar tópicos</li>
  <li>Atualização de tópicos</li>
  <li>Listagem de tópicos</li>
